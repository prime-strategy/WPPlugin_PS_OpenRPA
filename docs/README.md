## SVN

WordPress.org では SVN でソースコードが管理されています。

[SVN ソースコード](https://plugins.svn.wordpress.org/ps-openrpa) はこちらです。  
GitHub の最新リリースが自動的に公開されますので、基本的にはこちらの GitHub リポジトリでソースコードは管理されています。
